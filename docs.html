Getting Started

Welcome to PortfolioPro, a comprehensive portfolio management platform designed for professional investors. This guide will help you get started with all the features and tools available.

Quick Setup

  1. Upload your portfolio data using CSV or Excel files
  2. Set your cash balance for accurate portfolio valuation
  3. Configure target prices for your holdings
  4. Start tracking your transactions and performance

Portfolio Management

Uploading Portfolios

PortfolioPro supports multiple file formats for portfolio data:

  • CSV files: Comma-separated values with headers
  • Excel files: .xlsx and .xls formats
  • JSON files: Structured portfolio data

Required Columns

Symbol, Quantity, Price, Date, Total Cost

Portfolio Analysis

The platform provides comprehensive analysis including:

  • Asset allocation charts
  • Performance tracking over time
  • P&L calculations
  • Return percentage analysis
  • Portfolio comparison tools

Stock Calculator

The Stock Calculator helps you perform various financial calculations for your investments.

Available Calculations

  • Profit/Loss: Calculate gains or losses on stock positions
  • Return Percentage: Calculate percentage returns
  • Dollar Cost Averaging: Analyze DCA strategies
  • Portfolio Valuation: Calculate total portfolio value

Target Price Management

Set and track target prices for your holdings to monitor performance against your investment goals.

Setting Target Prices

  2. Select a portfolio from the dropdown
  3. Set target prices for individual stocks
  4. Save your targets for future reference

Troubleshooting

Common Issues

File Upload Issues

Ensure your CSV/Excel files have the correct column headers and data format. Check that all required fields are present and data is properly formatted.

Data Not Loading

Try refreshing the page or clearing your browser cache. If the issue persists, check your internet connection.

Performance Issues

For large portfolios, consider breaking them into smaller files or using the bulk upload feature for better performance.
